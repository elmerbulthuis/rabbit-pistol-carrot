What is the gap?

There is a gap between user stories and the actual implementation. This gap could be closed via SbE.

Enrich the user stories with concrete examples that can be turned into automated tests. These examples become a single source of truth that is very difficult to misunderstand. Because they can be testes automatically they are always up to date.
